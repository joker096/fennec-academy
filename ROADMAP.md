# 🚀 FlipLang Evolution Roadmap

This document outlines the planned improvements for FlipLang, inspired by modern language learning platforms and the Fallout universe.

## Phase 1: Core Learning Enhancements (Current Focus)

### 1. 📉 Mistakes Bank: "The Broken Terminal"
*   **Concept:** A dedicated area to practice words where the user has made mistakes.
*   **Fallout Theme:** A "corrupted" terminal that needs "repairing" by answering correctly.
*   **Mechanics:** 
    *   Automatically track failed words in lessons/flashcards.
    *   Require 3 consecutive correct answers to "repair" a word and remove it from the bank.
    *   Bonus XP/Caps for clearing the bank.

### 2. 🎙️ Speaking Practice: "Voice Calibration"
*   **Concept:** Use Web Speech API to verify user pronunciation.
*   **Fallout Theme:** Calibrating the Pip-Boy's voice recognition module.
*   **Mechanics:**
    *   Add a microphone button to flashcards and lessons.
    *   Compare user input with the target word.
    *   Reward "Perfect Calibration" with extra XP.

## Phase 2: Engagement & Retention

### 3. 🏆 Vault Divisions (Leagues)
*   **Concept:** Tiered leaderboard system.
*   **Fallout Theme:** Moving from "Surface Scavenger" to "Vault Overseer" divisions.
*   **Mechanics:**
    *   Weekly resets.
    *   Top 3 users move up a division; bottom 3 move down.
    *   Higher divisions grant multipliers for earned Caps.

### 4. 📖 Overseer Logs (Interactive Stories)
*   **Concept:** Branching narrative dialogues.
*   **Fallout Theme:** Reading holotapes of past survivors and choosing how to respond.
*   **Mechanics:**
    *   Pre-written scripts in the target language.
    *   Comprehension questions embedded in the story.
    *   Unlockable lore about the FlipLang wasteland.

## Phase 3: Advanced Features & Polish

### 5. ✍️ Writing Systems Trainer: "Matrix Decryption"
*   **Concept:** Specific practice for non-Latin scripts (Japanese, Chinese, etc.).
*   **Fallout Theme:** Decrypting encrypted data packets.
*   **Mechanics:**
    *   Character matching (Sound -> Symbol).
    *   Stroke order visualization (if possible).
    *   Focus on Hiragana/Katakana and basic Hanzi.

### 6. 🎨 UI Customization: "Pip-Boy Skins"
*   **Concept:** Allow users to change the interface color scheme.
*   **Fallout Theme:** Switching between different Pip-Boy models (Classic Green, Amber, White, Blue).
*   **Mechanics:**
    *   Purchase skins using earned Caps.
    *   Toggle between themes in the Settings menu.
