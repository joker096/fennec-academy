# PolyglotPro: Vault-Tec Edition ☢️

Добро пожаловать в документацию проекта! Этот файл описывает структуру, логику и архитектуру нашего приложения для изучения языков, стилизованного под **Fallout 76**.

## 🏗 Структура проекта

Проект построен на **React (Vite)** + **Tailwind CSS** + **Zustand** (для управления состоянием).

### Основные директории и файлы:

*   **`/src/App.tsx`**
    Главный компонент приложения. Здесь настроен роутинг (`react-router-dom`), глобальный Layout (боковое меню, верхняя панель) и тематический фон (CRT-эффекты, сканлайны, индустриальный бункер).
*   **`/src/index.css`**
    Глобальные стили. Здесь подключены шрифты (Oswald, Roboto, VT323) и настроены кастомные CSS-классы для анимаций (например, `.scanlines`, `.crt`).
*   **`/src/store/useStore.ts`**
    Глобальное хранилище состояния (Zustand). Хранит:
    *   `xp` (Опыт) и `level` (Уровень вычисляется как `Math.floor(xp / 100) + 1`).
    *   `stimpaks` (Жизни/Стимпаки).
    *   `daysSurvived` (Серия дней/Streak).
    *   `targetLang` (Изучаемый язык).
    *   `equippedPerks` (Экипированные карточки перков).
*   **`/src/data/gameData.ts`**
    База данных приложения (в будущем можно перенести в Supabase). Содержит:
    *   `FALLOUT_PERKS` — список карточек перков (с оригинальными картинками, требованиями к уровню и эффектами).
    *   `LANGUAGES` — доступные языки.
    *   `SERBIAN_COURSE` — структура уроков (юниты, статусы).
    *   `SERBIAN_WORDS` — словарь с переводами и транскрипциями.

### Страницы (`/src/pages/`):

1.  **`Dashboard.tsx` (Маршрут `/`)**
    Главный экран обучения. Отображает модули (Юниты) в виде пути. Пользователь кликает на доступный урок, чтобы начать тренировку.
2.  **`Flashcards.tsx` (Маршрут `/flashcards`)**
    Интервальное повторение. Карточки со словами, которые можно переворачивать. Есть кнопка аудиопроизношения (использует `SpeechSynthesis API`).
3.  **`Perks.tsx` (Маршрут `/perks`)**
    Коллекция карточек перков из Fallout 76. Карточки разблокируются при достижении определенного уровня (XP). Игрок может экипировать до 3 карточек для получения бонусов.
4.  **`Lesson.tsx` (Маршрут `/lesson/:id`)**
    Интерактивный урок (квиз). Пользователь должен выбрать правильный перевод слова. При ошибке тратится Стимпак. При успехе начисляется XP.
5.  **`AIChat.tsx` (Маршрут `/chat`)**
    Чат с ИИ (Gemini API), который отыгрывает роль **Overseer AI** (Смотрителя Убежища). Помогает с языковой практикой и дает советы по выживанию.
5.  **`AdminSettings.tsx` (Маршрут `/admin`)**
    Панель администратора для управления пользователями, рекламой, аналитикой и глобальными настройками приложения. Доступ ограничен ролью `admin`. Подробности см. в [ADMIN_GUIDE.md](./ADMIN_GUIDE.md).

## ⚙️ Основные механики

1.  **Прокачка (Leveling):**
    Прохождение уроков (`Lesson.tsx`) дает XP. Каждые 100 XP повышают уровень.
2.  **Перки (Perk Cards):**
    На определенных уровнях (например, 28 уровень для *Arms Keeper*) открываются карточки. Они дают пассивные бонусы (визуально описаны в `gameData.ts`).
3.  **Аудио и Транскрипция:**
    В карточках и уроках используется встроенный синтезатор речи браузера для озвучивания слов на изучаемом языке.
4.  **Стимпаки (Здоровье):**
    Ошибки в уроках отнимают Стимпаки. Если они закончатся, урок будет провален (логика расширяема).

## 🎨 Дизайн и UI

*   **Шрифты:**
    *   `Oswald` (Display) — для заголовков и названий на карточках (имитирует узкий жирный шрифт Fallout).
    *   `Roboto` (Sans) — для читаемого основного текста.
    *   `VT323` (Mono) — для терминалов и транскрипций.
*   **Тематика Vault-Tec:**
    Используются фирменные цвета: глубокий синий (`#003b71`), желтый (`#f4cb38`), а также текстуры старой бумаги (`#e3dccf`) для карточек. Добавлены эффекты старого монитора (scanlines).
