# Руководство по проекту Fennec Brotherhood

## О проекте
**Название:** Fennec Brotherhood
**Описание:** Платформа для изучения языков с ИИ-наставником, интерактивными историями и алгоритмами интервального повторения (SRS) в стилистике постапокалипсиса (Fallout).

### Технологический стек
*   **Frontend:** React 19, TypeScript, Vite
*   **Стилизация:** Tailwind CSS 4
*   **Анимации:** Motion (Framer Motion)
*   **Состояние:** Zustand
*   **База данных и Аутентификация:** Firebase (Firestore & Auth)
*   **Искусственный интеллект:** Google Gemini SDK (@google/genai)
*   **Иконки:** Lucide React
*   **Визуализация данных:** Recharts, D3.js

---

## Основные возможности
1.  **AI Tutor (ИИ-наставник):** Автоматическая генерация объяснений ошибок, отзывов о произношении и грамматических разборов.
2.  **Интерактивные истории:** Ветвящиеся диалоги, где выбор пользователя влияет на исход и зависит от характеристик персонажа (S.P.E.C.I.A.L.).
3.  **Система SRS:** Умные флеш-карточки для запоминания слов с учетом прогресса.
4.  **Геймификация:** Уровни, опыт (XP), валюта (Крышки) и перки.
5.  **Task List:** Список задач с визуальным отслеживанием прогресса.
6.  **Knowledge Graph:** Визуализация связей между изученными концепциями.

## Локальная разработка

1.  **Клонирование репозитория** (после загрузки на GitHub):
    ```bash
    git clone <url-репозитория>
    cd fennec-brotherhood
    ```

2.  **Установка зависимостей:**
    ```bash
    npm install
    # Перед запуском скачайте аудио-ресурсы (выполняется автоматически при npm run dev)
    npm run download-audio
    ```

3.  **Настройка окружения:**
    Создайте файл `.env` на основе `.env.example` и добавьте ваши ключи (Gemini API Key и др.).

4.  **Запуск сервера разработки:**
    ```bash
    npm run dev
    ```

## Деплой (Развертывание)

### 1. AI Studio (Cloud Run)
Самый простой способ — использовать кнопку **Deploy** прямо в интерфейсе AI Studio. Это автоматически развернет приложение в Google Cloud Run.

### 2. Статический хостинг (Vercel / Netlify / Firebase Hosting)
Приложение является SPA (Single Page Application).
1.  Соберите проект:
    ```bash
    npm run build
    ```
2.  Загрузите содержимое папки `dist` на любой хостинг статических сайтов.

### 3. Firebase (Правила безопасности)
Если вы вносите изменения в структуру данных, не забудьте обновить правила:
```bash
# В AI Studio есть инструмент для этого, либо через Firebase CLI:
firebase deploy --only firestore:rules
```

## Инструкция по загрузке на GitHub

Если вы хотите сохранить проект в своем GitHub аккаунте:

1.  **Создайте новый репозиторий** на [github.com](https://github.com/new). Не инициализируйте его README или .gitignore.
2.  **Инициализируйте локальный репозиторий** в терминале AI Studio (если еще не сделано):
    ```bash
    git init
    git add .
    git commit -m "Initial commit: Fennec Brotherhood Project"
    ```
3.  **Привяжите удаленный репозиторий:**
    ```bash
    git remote add origin https://github.com/ВАШ_ЛОГИН/НАЗВАНИЕ_РЕПО.git
    ```
4.  **Отправьте код:**
    ```bash
    git branch -M main
    git push -u origin main
    ```

**Важно:** Убедитесь, что ваш `.gitignore` содержит `node_modules`, `dist` и файлы со секретами (`.env`).
