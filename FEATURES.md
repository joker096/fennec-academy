# Документация по функциям и механикам приложения (FlipLang)

## Философия названия: FlipLang
*   **Flip (Переворот):** Основная механика — переворот карточки. Это простое, понятное действие, которое лежит в основе процесса обучения.
*   **Lang (Язык):** Четкое указание на сферу приложения.
*   **Запоминаемость:** Короткое, двусложное, ритмичное. Такие названия хорошо работают в сарафанном радио: "скачай FlipLang" произносится одним дыханием.
*   **Международность:** Слово "flip" понятно носителям английского, но звучит современно и для русскоязычной, и для европейской аудитории. Суффикс "Lang" (от language) добавляет прозрачность без лишних слов.
*   **Маркетинговый потенциал:** Название само генерирует слоганы: "Flip your way to fluency", "One flip — one word", "Flip it, learn it". Это сильно упрощает продвижение.

Этот документ содержит структурированное описание всех текущих функций, опций и игровых механик приложения. Он послужит основой для создания полноценного руководства пользователя (инструкции).

## 1. Основная концепция и интерфейс
*   **Тематика:** Постапокалипсис в стиле Fallout (ретро-футуризм, терминалы RobCo, Пип-Бой).
*   **Интерфейс:** Стилизован под ЭЛТ-мониторы (CRT-эффекты, зеленый/янтарный текст на черном фоне, сканлайны).
*   **Подсказки (Tooltips):** Современные, привлекательные всплывающие подсказки с темной темой, плавными анимациями и адаптивной шириной, улучшающие навигацию (например, быстрый переход к модулям на главном экране).
*   **Языки:** Поддержка изучения нескольких языков (Английский, Сербский, Испанский, Французский, Немецкий, Итальянский, Японский, Китайский) с возможностью выбора языка интерфейса (Английский, Русский).

## 2. Игровая механика и прогрессия (Базовые параметры)
*   **Опыт (XP) и Уровни:** Начисляются за правильные ответы, прохождение уроков, мини-игры и квесты. Каждые 100 XP повышают уровень персонажа (Resident).
*   **Крышки (Caps):** Внутриигровая валюта. Зарабатывается за прохождение уроков, квестов и мини-игр. Тратится на покупку Перков (Способностей).
*   **Здоровье (HP):** Максимум 100 HP. Ошибки в уроках наносят урон (по умолчанию -20 HP). Если HP падает до 0, персонаж "умирает", и счетчик дней выживания (серия) сбрасывается.
*   **Стимпаки (Stimpaks):** Аптечки для восстановления здоровья (+50 HP за использование). Выдаются при ежедневном входе или покупаются/находятся.
*   **Серия дней (Дни выживания) и Радиация:** За ежедневный вход дается бонус (XP и стимпаки). Пропуск дней наносит урон от "радиации" (-30 HP за каждый пропущенный день), что может привести к потере серии.

## 3. Динамическая система S.P.E.C.I.A.L.
Характеристики персонажа (от 1 до 10), которые прокачиваются в зависимости от действий пользователя в приложении. При повышении появляется всплывающее уведомление в стиле Fallout.
*   **Strength (S) - Сила:** Базовый параметр (в резерве для будущих механик).
*   **Perception (P) - Восприятие:** Есть шанс (10%) повысить при идеальном повторении карточки (оценка 5 - "Отлично помню").
*   **Endurance (E) - Выносливость:** Есть шанс (10%) повысить при ежедневном входе в приложение (поддержание серии выживания).
*   **Charisma (C) - Харизма:** Повышается на +1 (плюс 50 XP и 20 крышек), когда вы успешно завершаете ролевой сценарий в чате с ИИ (ИИ выдает скрытый триггер `[SUCCESS]`).
*   **Intelligence (I) - Интеллект:** Есть шанс (30%) повысить при идеальном прохождении урока (без единой ошибки).
*   **Agility (A) - Ловкость:** Есть шанс (30%) повысить при успешной игре в "Match Madness" (если набрано больше 5 очков).
*   **Luck (L) - Удача:** Базовый параметр (в резерве для будущих механик).

## 4. Перки (Способности)
Покупаются за Крышки в специальном меню. Дают пассивные бонусы:
*   **Lead Belly (Свинцовое брюхо):** 30% шанс избежать получения урона при ошибке.
*   **Starched Genes (Крахмальные гены):** Если HP падает до 0, серия дней (streak) не сбрасывается.
*   **Inspirational (Вдохновляющий):** Увеличивает ежедневный бонус за вход (52 XP вместо 50 XP).
*   **Action Boy/Girl (Живчик):** Дает +2 Стимпака при ежедневном входе вместо +1.
*   **Concentrated Fire (Массированный огонь):** Убирает 1 неправильный вариант ответа в тестах (оставляет 3 варианта вместо 4).

## 5. Учебный процесс (G.O.A.T. Экзамены / Уроки)
*   **Сектора (Темы):** Уроки разбиты на сектора (Основы, Путешествия, Еда, Фонетика и т.д.). Для открытия следующего сектора нужно пройти предыдущий.
*   **Типы вопросов:**
    *   Выбор перевода на родной язык.
    *   Выбор перевода на изучаемый язык.
    *   Аудирование (восприятие на слух).
    *   Ввод текста (написание слова).
        *   **Динамический банк символов (Character Bank):** Для заданий на ввод текста автоматически генерируются кнопки с нужными символами и дистракторами. Это позволяет легко вводить специфические символы (иероглифы, умлауты, акценты) без необходимости переключать раскладку клавиатуры ОС.
    *   Фонетика (выбор правильной транскрипции - специально для Сектора 7).
*   **Комбо-система:** Серия правильных ответов подряд увеличивает множитель получаемого XP.
*   **Напарник (Фенек / Fennec):** 
    *   Радуется при наборе комбо.
    *   Может дать подсказку 1 раз за вопрос (убирает неверный ответ или открывает первые 2 буквы при вводе текста).

## 6. Карточки для запоминания (Flashcards / Банк памяти)
*   **Интервальное повторение (SRS 2.0):** Улучшенный алгоритм SuperMemo-2, который теперь учитывает:
    *   **Время суток:** Утром интервалы слегка увеличиваются (мозг свежий), ночью — уменьшаются (выше шанс забыть).
    *   **Усталость (Fatigue):** Алгоритм отслеживает количество карточек, пройденных за текущую сессию. Если пользователь устал (более 20-30 карточек), штрафы за ошибки возрастают, а интервалы для правильных ответов сокращаются, чтобы гарантировать лучшее закрепление материала.
*   **Режим Викторины (Quiz Mode):** Интерактивный режим проверки знаний с выбором правильного ответа из нескольких вариантов.
    *   **Озвучка вариантов:** В режиме викторины пользователи могут прослушать произношение каждого варианта ответа перед выбором, нажав на иконку динамика.
*   **Озвучка:** Встроенный Text-to-Speech (TTS) для произношения слов на изучаемом языке.
*   **Разделение:** Вкладки "К повторению" (Review) и "Изученные" (Mastered).

## 7. Встречи в Пустоши (Wasteland Encounters / Ролевые сценарии)
Практика языка с ИИ-персонажами в реалистичных ситуациях.
*   **Локации и сценарии:**
    *   *Станция монорельса (Вокзал/Аэропорт):* Покупка билетов, навигация. ИИ — ворчливый кассир.
    *   *Торговый пост (Магазин/Рынок):* Торговля, цены, покупка стимпаков и воды. ИИ — суровый торговец.
    *   *Клиника Пустоши (Больница/Аптека):* Описание симптомов, лечение. ИИ — уставший доктор.
    *   *Местный бар (Ресторан/Бар):* Заказ еды, общение. ИИ — дружелюбный бармен.
*   **Механика:** Чат работает через Gemini API. ИИ общается ТОЛЬКО на изучаемом языке. При достижении цели сценария ИИ выдает триггер `[SUCCESS]`, что дает награду (Харизма, XP, Крышки).

## 8. Тренировочные симуляции
*   **Match Madness (Безумие совпадений):** Мини-игра на время (60 секунд). Нужно соединять пары слов (оригинал - перевод). Зарабатывает XP, Крышки и шанс прокачать Ловкость (Agility).
*   **Архивы Голозаписей (Видео-уроки):** Поиск обучающих видео на YouTube по выбранному языку. Есть готовые запросы (Алфавит, Грамматика, Диалоги и т.д.). Просмотр засчитывается в ежедневные квесты.

## 9. Ежедневные квесты (Daily Quests)
Обновляются каждый день. За выполнение даются Крышки.
1.  Заработать 50 XP.
2.  Повторить 10 карточек.
3.  Пройти 1 урок.
4.  Посмотреть 1 видео-урок.

## 10. Библиотека (Архивы данных)
*   Полный словарь всех доступных слов в приложении, разбитый по категориям.
*   Возможность вручную добавить любое слово в колоду карточек для интервального повторения.
*   Поиск по словарю.

## 11. SEO и Оптимизация (Технические детали)
*   **Динамические Мета-теги:** Использование `react-helmet-async` для генерации уникальных title, description и keywords для каждой страницы (Dashboard, Lesson, Flashcards, Scenarios и т.д.).
*   **Ленивая загрузка (Lazy Loading):** Маршруты приложения загружаются по частям (code splitting) с помощью `React.lazy` и `Suspense`, что ускоряет первоначальную загрузку.
*   **Мемоизация (Memoization):** Использование `useMemo` для кэширования тяжелых вычислений (например, фильтрация списков слов и курсов), чтобы избежать лишних перерисовок интерфейса.
*   **Динамическая загрузка контента:** Логика выборки слов для уроков переписана на динамическую, что позволяет бесконечно добавлять новые темы (Погода, Тело, Одежда и т.д.) без изменения основного кода уроков.

## 12. Roadmap (План развития)

### Высокий приоритет
*   🎙️ **Голосовые карточки:** Распознавание речи для проверки произношения. Оценка акцента по шкале 1–10 с рекомендациями.
*   📱 **Офлайн-режим + виджет:** Карточка дня на экране блокировки. Повторение без открытия приложения — снижает барьер.

### Средний приоритет
*   📖 **Contextual Examples:** For each word in the vocabulary, fetch and display 3 real-life sentences from news, books, or social media where the word is used in context. Ensure this integrates with the flashcard and lesson modules.
*   🖼️ **Визуальные карточки + AI-картинки:** Автогенерация иллюстраций через AI для абстрактных слов. Мнемоника через образы.
*   🗺️ **Карта знаний:** Визуальный граф слов по темам, степени освоенности и связям. Мотивирует заполнять «белые пятна».
*   🤖 **AI-собеседник:** Чат-бот для практики изученных слов в диалоге. Исправляет ошибки и предлагает синонимы.

### Базовый приоритет
*   📊 **Умная статистика:** Слабые места, лучшее время для повторения, прогноз сдачи экзамена. Инсайты, а не просто счётчик.
