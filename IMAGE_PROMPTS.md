# Image Generation Prompts Guide

This document contains a curated list of prompts for generating high-quality, consistent visual assets for the application. The style is inspired by modern educational apps (like Duolingo) but adapted to our **Post-Apocalyptic Wasteland** theme.

## Visual Style Guide: "Wasteland Vector-Plus"

To ensure consistency across all generated images, every prompt should include these style keywords:

> **Style Keywords:** *Vibrant 3D vector illustration, stylized flat design, clean bold lines, cinematic soft lighting, post-apocalyptic wasteland aesthetic, warm color palette (indigo, amber, emerald), high-quality digital art, educational and friendly tone, 8k resolution, trending on Behance/Dribbble.*

## Pro-Tips for Best Results:
1. **Aspect Ratio:** Use `--ar 1:1` for icons and `--ar 16:9` for hero banners.
2. **Negative Prompts:** Use keywords like `scary, dark, gritty, realistic, photo, messy, blurry, low resolution, text, watermark` to keep the style clean and app-friendly.
3. **Consistency:** Always keep the "Style Keywords" at the end of every prompt to maintain a unified look across the app.
